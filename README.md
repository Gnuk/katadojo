# Kata Dojo

This is a basic command to generate dojo from template.

## Prerequisites

* [Node](https://nodejs.org/) LTS

## Install

```shell
npm i -g katadojo
```

## Usage

To know how to use, please launch:

```shell
katadojo help
```

## Development

Run tests with:

```shell
npm test
```

Build with:

```shell
npm run build
```

Install locally with:

```shell
npm install --global
```
